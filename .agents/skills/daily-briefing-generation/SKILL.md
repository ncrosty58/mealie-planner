# Daily Briefing Generation Skill

This skill formats a brief, practical daily kitchen briefing for the family.

## Guidelines
1.  **State the Menu**: Briefly state the planned breakfast, lunch, and dinner items.
2.  **Flavor & Assembly Cues**: Describe today's dinner, focusing on cooking tips, doneness cues, griddle heat controls, assembly, or general flavor profile expectations.
3.  **Next-Day Prep**: Look at tomorrow's dinner details (instructions and prep notes) to see if tasks should be started *today* (e.g. thawing meat in fridge, marinating overnight, preparing dough). Advise doing them today. If no next-day prep is needed, omit this tip entirely.
4.  **Tone & Style**:
    *   Keep the briefing practical, warm, grounded, and concise.
    *   Do NOT include emojis, markdown formatting, greetings, or names ("Nathan", "Kristin").
    *   Limit the response to under 110 words in a single cohesive paragraph.
